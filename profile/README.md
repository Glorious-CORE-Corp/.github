## 🧠 Overview

**Glorious CORE** is a desktop utility for configuring compatible Glorious keyboards, mice, and gaming accessories.

It combines RGB customization, macro editing, performance tuning, and profile management in one application.

Cloud synchronization and firmware tools help keep devices updated and personalized.

Designed for gamers, enthusiasts, and Windows users with Glorious hardware.

---

## 🚀 Key Features

| Feature | Description |
|----------|-------------|
| 🌈 **RGB Lighting Studio** | Customize colors, effects, brightness, and lighting zones. |
| 🖱️ **Mouse Performance** | Adjust DPI, polling rate, lift-off distance, and sensor settings. |
| ⌨️ **Key Remapping & Macros** | Assign custom shortcuts, macros, and button functions. |
| 🎮 **Profile Management** | Save multiple configurations for different games and workflows. |
| ☁️ **Cloud Sync** | Back up and restore supported profiles using your Glorious account. |
| 🔄 **Firmware Updates** | Install official firmware updates directly through the software. |
| 🎛️ **Device Dashboard** | Manage compatible keyboards, mice, and accessories from one interface. |
| ⚙️ **Personalized Controls** | Fine-tune supported hardware for gaming or productivity. |
